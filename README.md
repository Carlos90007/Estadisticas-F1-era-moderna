# Estadisticas-F1-era-moderna
Se trata de un trabajo en R, analizando diferentes equipos de F1, sus tiempos de parada en boxes, como afecta la posición de salida y el orden de llegada a la meta en cada carrera y los diferentes tiempos de parada en boxes en función del circuito. Vemos cada caso a través de Boxplots, histogramas, Violin Plot, una matriz de correlación
